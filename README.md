# Patient Health Database

Author: Daniel Gan

## Introduction
This repository contains code for running a patient health database.  
This is written in python.

## Sepcifications
The database will contain:  
* Patient Name
* Patient Medical Record Number
* Patient Age
* Patient Test Results



* Patient Bloodtype
	- A+
	- A-
	- B+
	- B-
	- AB+
	- AB-
